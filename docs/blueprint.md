# **App Name**: LexiAI

## Core Features:

- Document Upload: Allow users to upload legal documents (PDF, DOCX, TXT) to the application.
- AI-Powered Chat: Enable users to ask questions about the uploaded documents and receive answers powered by a large language model (LLM).
- Contextual Understanding: The LLM should leverage tool use and other methods to effectively process documents, including tables, citations, and legal jargon, and cross-reference information.
- Suggested Questions: Provide users with suggested follow-up questions based on their previous queries and the document content.
- Clear Presentation: Display the answers in a well-formatted and easy-to-understand manner.
- Session Storage: Save the user documents and the converstation using a postgres database
- User Interface: Easy-to-use layout.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to inspire trust and professionalism.
- Background color: Light gray (#F3F4F6) to reduce eye strain and maintain a clean interface.
- Accent color: Teal (#2DD4BF) for interactive elements and important highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look; suitable for headlines or body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use professional and clear icons related to legal documents and AI features.
- Clean, structured layout with a focus on readability and ease of navigation.
- Subtle transitions and animations to improve user experience without distracting from content.
